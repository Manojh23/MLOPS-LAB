# Experimental Tracking Lab

This folder contains a **combined notebook** prepared as part of the **Experimental Tracking Lab**.

## Included files
- `Combined_Experimental_Tracking_Lab.ipynb` - one combined notebook with outputs already shown below each code cell
- `outputs/` - saved metrics JSON files and trained model files from the runs

## What is different in this version?
This version uses a **different dataset** and a **different model** from the original notebooks:
- **Dataset:** Breast Cancer Wisconsin dataset from `scikit-learn`
- **Model:** `RandomForestClassifier`

## Notes
- The notebook is designed as a simple combined lab file.
- Experiment tracking is done in **W&B offline mode** so the notebook can still run locally.
- The main purpose is to keep this as part of the **Experimental Tracking Lab** in a simple format.
